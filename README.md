# 🔐Генератор паролей

Простой консольный генератор паролей на **Java**.<br>
Пароль создаётся на основе названия сервиса и пользовательского числа, что позволяет получать **одинаковый пароль для одного и того же сервиса**. 
--- 


## ✨Возможности

- Создание паролей

---
## 🛠️ Технологии

- Java 21+
- Никаких внешних зависимостей
