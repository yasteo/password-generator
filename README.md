# 🔐Генератор паролей

Простой консольный генератор паролей на Java.<br>
Пароль создаётся на основе названия сервиса и пользовательского числа, что позволяет получать одинаковый пароль для одного и того же сервиса. 
--- 


## ✨Возможности

- Создание паролей

---
## 🛠️ Технологии

- Java 21+
- Никаких внешних зависимостей

---

## 📁 Структура проекта

```
src/
├── Main.java
├── Generator.java
├── FileReader.java
└── UniqueGenerator.java
```

---

## ⚙️ Запуск проекта

```bash
git clone git@github.com:yasteo/password-generator
cd password-generator
javac src/*.java
java -cp src Main
```

---

## 📌 Пример работы

```
Введите название сервиса: Steam
Введите число: 333
Ваш пароль: (тут пароль)
Хотите продолжить? [Y/N]: 
```
